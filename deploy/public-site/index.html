INTENT TO CONTRACT

Say the job in plain English. NODE turns it into operating rules.

Example request: "Fix the backend bug, but do not touch frontend files." NODE translates that intent into explicit permissions, budget limits, and approval rules a daemon can enforce.

Human intent

Fix backend only.

The user does not need to write YAML first. The important part is that the final contract is inspectable before the run starts.

NODE control contract 
task: Fix backend bug. Do not touch frontend.
capabilities:
  filesystem_read: ["/workspace"]
  filesystem_write: ["/workspace/internal"]
  network_destinations: []
  secrets: []
budget:
  max_tokens: 120000
  max_cost_usd: 5
approval_rules:
  - action: fs.write
    match: /workspace/internal
