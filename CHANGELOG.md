# Changelog

## v1 Local Developer Preview

- Added local-first daemon, CLI, dashboard, SQLite event store, replay, audit export, approvals, budgets, and Docker worker execution.
- Added OpenAI Agents SDK adapter and offline coding-agent example.
- Added Windows state ACL hardening for state directory, operator token, and SQLite files.
- Added `agentos doctor` for local preflight checks.
- Added `agentos validate <manifest.yaml>` for manifest and local environment validation.
- Added `agentos version` with build metadata.
- Added deployment, scaling, security, contributing, and release-readiness docs.
- Updated Windows packaging to ship the full local source/runtime kit with archive hygiene checks.
