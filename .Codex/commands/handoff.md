# /handoff <agent>

Prepare a bounded handoff to a specialist.

1. Confirm the target exists in the kernel registry.
2. State objective, ownership boundary, relevant files, invariants, decisions,
   current evidence, and expected validation.
3. Include the current session ID and known blockers.
4. Ask the specialist to append its result to the same session, not create a
   disconnected history.
5. On return, reconcile conflicts and summarize the integrated outcome.
